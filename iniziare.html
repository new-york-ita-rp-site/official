Come iniziare

1. Entra nel Discord

Leggi annunci, regolamento, candidature e comunicazioni dello staff.


Apri Discord

2. Entra nel server Roblox

Usa il link del server e collegati a ERLC per iniziare il roleplay.


Apri Roblox

3. Scegli il tuo ruolo

Puoi essere cittadino, lavoratore, agente, soccorritore o criminale, rispettando sempre le regole.

4. Candidati per una fazione

Per NYPD, NYSO, ICE, EMS e altri reparti controlla le candidature nel Discord.

Il link vero del server va inserito in assets/js/script.js, variabile SERVER_LINK.
