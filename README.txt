Come cambiare link server, Discord, membri e versione:
1. Apri assets/js/script.js
2. Modifica queste righe:
   const SERVER_LINK = "https://www.roblox.com/games/INSERISCI-ID-GIOCO/NEW-YORK-ITA-RP";
   const DISCORD_LINK = "https://discord.gg/INSERISCI-LINK";
   const TOTAL_MEMBERS = "250+";
   const VERSIONE = "v1.0";

Come cambiare immagini/sfondi:
- Sfondi pagine: assets/img/backgrounds/
  home.jpg, fazioni.jpg, regolamento.jpg, storia.jpg, shop.jpg, iniziare.jpg

- Immagini fazioni: assets/img/factions/
  procura.jpg, ice.jpg, nypd.jpg, nyso.jpg, border-patrol.jpg, nyfd.jpg,
  ems.jpg, dot.jpg, aziende.jpg, gang.jpg, mafie.jpg

Per sostituire un'immagine:
- tieni lo stesso nome del file
- usa formato .jpg
- ricarica la pagina

Colori:
- Il sito è già blu.
- Per cambiare tonalità modifica :root in assets/css/style.css

Consiglio:
Apri index.html nel browser per vedere il sito.
